# Changelog

รูปแบบอิงจาก [Keep a Changelog](https://keepachangelog.com/) แบบคร่าวๆ เรียงจากใหม่ไปเก่า จัดกลุ่มตาม commit ที่ bump เวอร์ชัน (`id="versionLabel"` ใน `index.html`) ส่วนที่ไม่มี commit bump เวอร์ชันชัดเจนจะรวมไว้ในกลุ่มเวอร์ชันถัดไปที่ตามมา

## v15.0

- ปรับ layout หน้า home ใหม่เป็นแบบ sectioned list กดปุ่มเพื่อเล่นแทนการ์ดแบบ tile grid (มีการทดลอง tile-grid ก่อนหน้านี้แล้ว revert กลับ)
- แก้ layout ชื่อ/คำอธิบายเกมในแถว home ให้อยู่บรรทัดเดียวกันสำหรับชื่อเกมสั้นๆ
- เพิ่มภาพประกอบตัวละคร (character portrait) สำหรับบทบาทในคืนสมิง
- เปลี่ยนโลโก้เป็นมาสคอตใหม่ (logo1/logo2) พร้อม cache-busting URL
- เพิ่ม feedback ตอนกดปุ่มคัดลอกลิงก์ และปรับปรุง reconnect UX ของ Firebase ให้ฉลาดขึ้น
- เพิ่ม animation ตอนเปลี่ยนหน้าจอ และ animation ของรายชื่อผู้เล่นในล็อบบี้
- คงหน้าเดิมไว้เมื่อกด refresh, เพิ่มสีทีมในรายการบทบาทคืนสมิง, ปรับปรุง donate modal
- เพิ่มปุ่มเลี้ยงชาเขียว (donate) พร้อม modal แสดง QR PromptPay
- รีแบรนด์จาก Game Hub Teng เป็น WongPlay เปลี่ยนคำว่า "ห้อง" เป็น "วง" ทั้งแอป
- ปรับสีไอคอนหน้า home และปุ่มต่างๆ หลายรอบ (ตัด emoji outline, ปรับสี King's Game, mute/bright สลับไปมา)
- เปลี่ยนชื่อเกมเป็นภาษาไทย (Werewolf → คืนสมิง, Spyfall/Wheel/Cards เปลี่ยนชื่อ), แก้สีไอคอน
- แยก `index.html` ออกเป็นไฟล์ `css/` และ `js/` แยกต่างหาก (จากเดิมรวมไฟล์เดียว)
- ปรับ branding และ profile menu ของหน้า home
- รีดีไซน์ layout หน้า home และ popup ต่างๆ เปลี่ยนโลโก้แบบโปร่งใส
